# Physarum Evolution Notes

## Observed failure mode: ribbon domination

After running for a while, the simulation can collapse into broad diagonal highways with alternating cyan/coral bands and very little local diversity. The result reads more like a synthetic stripe field than interacting biological species.

That visual mode is now preserved as the `RIBBON DRIFT` preset so it can still be studied and compared against newer variants.

## What we want instead

- Biological-feeling species that form, split, survive, and interact
- Smaller colonies that do not get erased as soon as a dominant lineage appears
- Stronger visible mutation pressure and more ecological branching
- A bright petri-dish look with soft agar backgrounds and living colony colors
- A microscope mode where the viewer feels close to bacterial activity, not distant from abstract ribbons

## Current strategy

- Preserve multiple visual/ecology presets and switch them live with number keys
- Cap trail intensity so no lineage can build an unbeatable superhighway
- Increase mutation under crowding so dominant regions split into daughter clades
- Keep drift weak and kin-conditional so species interact without homogenizing
- Use light-field display modes for petri dish and microscope views
